# Plant Disease Detection System

## Introduction
This repository contains the code and documentation for a Plant Disease Detection System. It aims to help farmers and agricultural professionals identify plant diseases using machine learning and image processing techniques.

## Features
- Real-time disease detection
- User-friendly interface
- Support for multiple plant species
- Comprehensive disease database

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Pradeep7895/plant-disease.git
   ```
2. Navigate into the project directory:
   ```bash
   cd plant-disease
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Start the application:
   ```bash
   python app.py
   ```
2. Upload an image of the plant leaf.
3. The system will analyze the image and provide diagnosis of any detected diseases along with recommended actions.
